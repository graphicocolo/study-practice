# ルール

## 変数の名前

- 広い範囲で使うもの
  - 用途のわかる名前にする menu, element など
- 狭い範囲でしか使わない変数
  - 短い抽象的な名前でもOK arr, i, t など

## 関数

- 動詞 + 名詞 が一般的 getYear, setYear など
- 処理の内容にあった名前にする
- 一つの関数には一つの処理

## 進め方

- まず動くようにする
- 出力が正しくなくても構わない。その後手直しをして正しく動くようにする
- コードの構成は以下
  - HTML 要素取得
  - 必要データを定義もしくは API 経由でデータを取得
  - 各種関数を定義
  - イベントリスナーを定義
