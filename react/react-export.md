# React エクスポート

## コンポーネントとカスタムフックのエクスポート慣例

コンポーネントとカスタムフックで使い分けるのが慣例。

| | 慣例 | 理由 |
|--|------|------|
| コンポーネント | デフォルトエクスポート | 1ファイル1コンポーネントが多い |
| カスタムフック | 名前付きエクスポート | 1ファイルに複数フックを置くこともある、インポート時に名前が明確 |

```ts
// コンポーネント → デフォルトエクスポート
export default function BmiCalculator() { ... }

// カスタムフック → 名前付きエクスポート
export function useBmiCalculator() { ... }
export const useBmiCalculator = () => { ... }
```

強制ではなくコミュニティの慣例。
